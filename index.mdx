---
title: "lazygit"
description: "A simple terminal UI for git commands — making complex git operations accessible through an intuitive keyboard-driven interface."
---

lazygit is an open-source terminal UI for git that lets you stage files, manage branches, rebase interactively, cherry-pick commits, and much more — all without memorizing dozens of git commands.
